Política de Privacidade

Esta extensão não coleta, armazena nem compartilha quaisquer dados pessoais dos usuários.

Todos os dados manipulados pela extensão ficam restritos ao navegador do usuário e não são enviados a nenhum servidor externo.

A extensão foi desenvolvida com foco em privacidade, segurança e transparência, para uso interno por um órgão público.
